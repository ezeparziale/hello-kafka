# 🕸️ Server kafka

Docker compose listo para generar un server kafka con zookeeper para proyectos de streaming.

## 📦 Versiones

Tanto la versión de Zookeeper y Kafka son de Confluent 6.2.1

## 🏃‍♂️ Run

```bash
docker-compose -f "docker-compose.yaml" up -d
```
